THE SEVEN LAYERS OF OSI MODEL


The OSI Model (Open Systems Interconnection Model) is a conceptual framework used to describe the functions of a networking system. The OSI model characterizes computing functions into a universal set of rules and requirements in order to support interoperability between different products and software. There are advantages and disadvantages of osi model.
